# Web Extension

This repository contains browser extensions and web tools I built during my college days.
These work for the most part, but due to the nature of them being a product of my boredom, 
they may not be the most polished or efficient extensions out there.

Try them out and let me know if you find any bugs or have any suggestions. If you have the time and the energy, tinker around a bit and make some improvements or changes on your own.   

## Extensions & Plugins

| Extension Name | Version | Description | Folder |
| :--- | :--- | :--- | :--- |
| **Full-Page Screenshot** | 1.0 | Capture a full-page screenshot of any website, download as PNG, and copy to clipboard. | [webGrab](./webGrab) |
| **WA-Focus — WhatsApp Sidebar Toggle** | 1.0.0 | Toggle sidebar visibility and blur sidebar content on WhatsApp Web for distraction-free, full-width messaging. | [whatsappCollapse](./whatsappCollapse) |

---

## Getting Started

To install any of these extensions:
1. Open Chrome and go to `chrome://extensions/`.
2. Enable **Developer mode** (toggle in the top right).
3. Click **Load unpacked**.
4. Select the folder of the extension you want to install.
